Welcome to the Kanisa Project!

This is a project I have been working on for the past decade or so. There are two main parts; a generator and a renderer.

A generator will simply generate new Masterpiece files for use in the Renderer.

The renderer will create all sorts of different things with each Masterpiece file.

Masterpiece files are simple human-readable JSON files with specific keys and values for the renderer or a gen to use.
